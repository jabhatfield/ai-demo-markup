== Introduction

Handwriting Recogniser implements http://djl.ai[DJL,window=_blank] to train a neural network on images of handwritten numbers
within the range of 0 - 9. It can then generate the probabilities of an input image matching each of these numbers.

== Get started

See one or more of the following ways to get started, depending on preference of learning approach:

* Perform the xref:intro-component::install-and-run-quickstart.adoc[] to run Handwriting Recogniser.
* Complete the xref:tutorial/handwriting-recogniser-tutorial.adoc[] to gain practical experience of the API.
* Read the xref:handwriting-recogniser-deep-dive.adoc[] to gain a deeper theoretical understanding of the chatbot.
* See the xref:intro-component::api-spec.adoc[] for fine details of the API.
